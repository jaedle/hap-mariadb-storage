# HAP-MariaDB-Store

This is a store adapter using [MariaDB](https://mariadb.org/) for [hap](https://github.com/brutella/hap).

## Supported Databases

The following databases are supported and performed integration tests against.

### MariaDb

- 10.3
- 10.4
- 10.5
- 10.6
- 10.7

### Mysql

- 5.6
- 5.7
- 8.0
